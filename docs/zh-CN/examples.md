## 你现在该看哪个 example

- 想先本地跑通一遍：看 [openclaw_quickstart](../../examples/openclaw_quickstart/README.md)
- 想最低成本接真实 runtime：看 [openclaw_proxy_minimal](../../examples/openclaw_proxy_minimal/README.md)
- 想让 Python helper 自动带上下文：看 [openclaw_python_helper](../../examples/openclaw_python_helper/README.md)
- 已经在用 OpenAI Python SDK 风格代码：看 [openclaw_openai_wrapper](../../examples/openclaw_openai_wrapper/README.md)
- 想提升 session / run / request 的稳定关联：看 [openclaw_with_headers](../../examples/openclaw_with_headers/README.md)
- 想提高 retry / fallback / routing 的 branch 保真：看 [openclaw_with_semantic_contract](../../examples/openclaw_with_semantic_contract/README.md)
- 已经有 captured runs，只差导出训练数据：看 [export_to_async_rl](../../examples/export_to_async_rl/README.md)

## 怎么搭配文档使用

- guide 用来回答“什么时候该选这条路径”
- example README 用来回答“下一步到底执行什么”
- 当 prose 和 runnable snippet 有冲突时，以 example README 和 CLI 当前行为为准
